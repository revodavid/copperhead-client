# CopperHead Client

Web-based client for the CopperHead 2-player Snake game.

## Features

- HTML5 Canvas rendering
- WebSocket connection to CopperHead server
- Keyboard controls (Arrow keys or WASD)
- Single and two-player modes

## Usage

1. Start the CopperHead server (via Codespaces or locally)
2. Open `index.html` in a browser
3. Enter the server WebSocket URL
4. Select player (1 or 2) and game mode
5. Click "Ready" to start

## Controls

- **Arrow keys** or **WASD**: Move snake
- **Space**: Ready up / Restart
